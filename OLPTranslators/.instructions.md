# OLP Translator Rules

## Safety
- **DO NOT MODIFY**: `downloadStarter.py`, `downloader.py`, `uploadStarter.py`, `uploader.py` — these are E2 installation files.
- **DO NOT MODIFY**: E2 site-packages (`cenpydownload`, `cenpyolpcore`, `cenpylib`, `cenpyupload`, `cenpymath`, `cenpyunits`).
- **DO NOT** install new Python packages unless explicitly approved.

## Coding Conventions
- **Indentation**: Check existing file conventions for each vendor. Default is 3 spaces.
- **Logger**: Always a local variable — `logger = operator.GetLogOperator()`. Never store as instance attribute.
- **Class registration**: Every downloader module must define `DOWNLOAD_CLASS_NAME = "ClassName"` at module level.
- **No debugging prints**: Use E2 logger (`logger.LogDebug()`, `.LogInfo()`, `.LogWarn()`, `.LogError()`).

## Event Attribute Reading
- **Use generic `GetAttributes()` loop** for reading event data — it works for ALL event types (`LogicPort`, `SetResourcePort`, `WaitForResourcePort`, `Speed`, `Accuracy`, etc.).
- **Typed getters** (`GetBoolAttribute`, `GetIntegerAttribute`, `GetDoubleAttribute`, `GetStringAttribute`) are only needed inside `LogicPort` subtypes where the `SignalValue` data type varies (bool/int/float).
- **Two-level dispatch for signals**: First dispatch on `event.GetEventType()` (the EventType enum), then only for `EventType.LogicPort` dispatch on the `eventSubType` string from the `'EventType'` attribute.

## Position Data
- E2 positions are in **meters** — multiply by 1000 for millimeters.
- External prismatic axes are also in meters.
- Sort external axes by `joint.GetJointIndex()` before output.

## File Output
- Buffer all output into arrays (`self.Source`, `self.Data`, etc.), write in `WriteOutputFile`.
- Always call `operator.AddOutputFilePath()` in `CloseOutputFile` to register files with E2.

## Testing
- Use `Simple_Python_Translator.py` to dump the OLP tree as `.txt` for analysis before coding.
- Deterministic output: guard timestamps/usernames with test flags (`CPOST_TESTLAUF_CENIT`).

## Documentation
- If you change downloader behavior or output format, update `docs/ctrl-specific/<VENDOR>/current.md` and the relevant spec under `docs/ctrl-specific/<VENDOR>/spec/`.

## Download Lifecycle (Callback Order)

The E2 download starter calls these methods in this exact order:

```
1. __init__()
2. Initialize(operator)
3. OutputHeader(operator, controller)
4. CreateOutputFile(operator)
5. ProgramStart(operator, program)
   ├─ OperationGroupStart(operator, operationGroup)
   │  ├─ OperationStart(operator, operation)
   │  │  ├─ HandleMotion(operator, motion)
   │  │  │  ├─ eventsBefore → HandleEvent(operator, event, motion)
   │  │  │  ├─ OutputPositionData(operator, position)
   │  │  │  ├─ OutputMotionData(operator, motion, ...)
   │  │  │  └─ eventsAfter → HandleEvent(operator, event, motion)
   │  │  └─ OperationEnd(operator, operation)
   │  └─ OperationGroupEnd(operator, operationGroup)
6. ProgramEnd(operator, program)
7. WriteOutputFile(operator)
8. CloseOutputFile(operator)
```

## Event Handling

### Dispatch on EventType enum, not event.GetName()

```python
def HandleEvent(self, operator, event, motion):
   et = event.GetEventType()
   if et == EventType.Speed:
      ...
   elif et == EventType.Accuracy:
      ...
   elif et == EventType.LogicPort:
      self._handleLogicPort(operator, event)
   elif et == EventType.SetResourcePort:
      self._handleSetResourcePort(operator, event)
```

### Common Event Attributes

| Event | Key Attributes |
|-------|---------------|
| `Speed` | `Value` (float), `PathType` ("Contour"/"PointToPoint") |
| `Accuracy` | `Value` (float), `PathType`, `Criteria` ("On"/"Off"/"Distance") |
| `Acceleration` | `Value` (float), `PathType` |
| `TextEvent` | `Text` (str), `IsComment` (bool) |
| `Dwell` | `Value` (float, seconds) |
| `LogicPort` | `EventType` (str subtype), `SignalName`, `SignalNumber`, `SignalValue` — **single signal** |
| `SetResourcePort` | `ResourceName`, then **repeating groups**: `SignalName`, `SignalAddress`, `SignalNumber`, `SignalValue` |
| `WaitForResourcePort` | `ResourceName`, then **repeating groups**: `SignalName`, `SignalAddress`, `SignalNumber`, `SignalValue` |

### CRITICAL: SetResourcePort / WaitForResourcePort are multi-signal containers

A single event can carry multiple signals. Attributes repeat in groups — a simple scalar loop only captures the last one. Collect into a list:

```python
signals = []
current = {}
for attr in event.GetAttributes():
   n = attr.GetName()
   if n == 'SignalName':
      if current.get('SignalName'):
         signals.append(current)
      current = {'SignalName': attr.GetValue(), 'SignalAddress': '', 'SignalNumber': '', 'SignalValue': ''}
   elif n in ('SignalAddress', 'SignalNumber', 'SignalValue'):
      current[n] = str(attr.GetValue())
if current.get('SignalName'):
   signals.append(current)
```

## Position Output

```python
xyz = position.GetXYZ()               # meters — multiply by 1000 for mm
angles = position.GetOrientation()    # degrees (Rx, Ry, Rz) — use as-is
x_mm, y_mm, z_mm = [v * 1000 for v in xyz]

# External axes — sort by index, convert prismatic m→mm
externals = sorted(position.GetExternalJointValues(), key=lambda e: e[0].GetJointIndex())
for joint_obj, value in externals:
   if joint_obj.GetJointType() == JointKinematicType.Prismatic:
      value = value * 1000
```

## File Output Pattern

```python
def __init__(self):
   self.Source = []          # buffer all output here

def WriteOutputFile(self, operator):
   FileUtility().AppendTextArrayToFile(self.outputPath, self.Source)

def CloseOutputFile(self, operator):
   operator.AddOutputFilePath(self.outputPath)   # REQUIRED — registers with E2
```

## Common Mistakes

1. **SetResourcePort multi-signal** — single scalar loop only captures the last signal; always collect into a list.
2. **Typed getters instead of generic loop** — `event.GetBoolAttribute()` crashes on wrong type; use `GetAttributes()` first.
3. **Forgetting `AddOutputFilePath()`** — file won't appear in E2.
4. **Wrong units** — E2 XYZ in meters; multiply by 1000. External prismatic axes too.
5. **Logger wrong scope** — `logger = operator.GetLogOperator()` must be a local variable; not available without `operator`.
6. **`GetIndex()` > 128** — Tool/Base profile not mapped; log an error, don't crash.
